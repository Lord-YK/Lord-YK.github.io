# Random [Gift]!

Enter a [Seed] and get a [Gift] back!

Infinite (2^256) possibilities!

(Pseudorandom, same seed same output)

## 1. Flower Field!

This a pseudo(?) random flower field generator.

Plants a few different flowers around the field (the screen).

Then, the flowers propagate such that the same seed produces the same resultant flowermap.

I think it's pretty cool.

Try out different inputs and see what beautiful flower fields you get back!
