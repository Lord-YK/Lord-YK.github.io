# Flower Field!

This a pseudo(?) random flower field generator.

First, it takes in your input and uses that as a seed for the mathdotrandom() function.

Then, it plants a few different flowers around the field (the screen).

The flowers propagate such that the same seed produces the same resultant flowermap.

I think it's pretty cool.

Try out different inputs and see what beautiful flower fields you get back!
